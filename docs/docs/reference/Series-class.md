# `Series-class`

Create new Series


## Description

Found in api as pl$Series named Series_constructor internally


## Format

An object of class `environment` of length 3.


## Usage

```r
Series(x, name)
```


## Arguments

Argument      |Description
------------- |----------------
`x`     |     any vector
`name`     |     string


## Examples

```r
pl$Series(1:4)
```
