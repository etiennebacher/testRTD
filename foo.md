# `DataFrame-get_column`

Extract columns from a Polars DataFrame


## Description

Extract columns from a Polars DataFrame


## Usage

```r
<DataFrame>$get_column(...)
<DataFrame>$get_columns(...)
```


## Arguments

Argument      |Description
------------- |----------------
`...`     |     Not used


## Examples

```r
DataFrame$new()$get_column("a")
DataFrame$new()$get_columns("a", "b")
```
